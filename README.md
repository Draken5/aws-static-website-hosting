# AWS Static Website Hosting

## Overview
This project demonstrates hosting a static website using Amazon S3.

## AWS Services Used
- Amazon S3
- IAM (for bucket permissions)

## Steps
1. Created an S3 bucket for the website
2. Enabled static website hosting on the bucket
3. Uploaded HTML, CSS, and JS files
4. Configured public access and permissions
5. Verified the website loads correctly via the S3 endpoint

## Architecture
User → S3 Static Website

## Key Learnings
- How to host a static website in the cloud
- Configuring S3 permissions for public access
- Basic cloud architecture concepts
